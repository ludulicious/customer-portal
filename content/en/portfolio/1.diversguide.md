# divers-guide.com - Your Ultimate Diving Companion

divers-guide.com is a comprehensive platform designed for diving enthusiasts of all levels. Whether you're a beginner looking for your first dive or an experienced diver seeking new adventures, our platform connects you with the best diving experiences worldwide.

![diversgide website](/images/portfolio/divers-guide/main.png){.rounded-lg height="600" width="900"}

## Key Features

- **Interactive Dive Spot Map**: Explore thousands of dive sites with detailed information, difficulty levels, and user reviews
- **Dive Spot wiki**: Dive Spots maintained an categorized by a community of enthousastic divers
- **Dive Log Integration**: Track your dives, log your experiences, and maintain a digital dive journal
- **Search dive logs**: Search the most recent dive logs for favourite spots with attributes like visibility and water temperature

## Technical Implementation

We built the Diversguide API running Asp.Net Core, using the C# programming language.  We have built the user portal where divers can log their dives, propose changes to dive spots aand where admins can easily moderate the proposed changes. 

We also built a app for iOS and Android that connects to the API.

Built with modern web technologies, divers-guide.com offers a seamless user experience across all devices. The platform features:

- **Responsive Design**: Optimized for desktop, tablet, and mobile devices
- **User Authentication**: Secure login system
- **Content Management**: Easy-to-use admin panel for content updates

## Impact

Since launch, divers-guide.com has helped over 15,000 divers discover new dive spots and connect with the global diving community. The platform has received positive feedback for its user-friendly interface and comprehensive resource library.
