# Gratisaftehalen - Place adds for free to give away the stuff you don't need anymore

Gratisaftehalen.nl is a comprehensive Dutch marketplace platform that allows users to place an ad for free for the stuff that they
don't have any use for anymore, bit is too good to throw away.

Other users can browse these adds and make arrangements for the items to be picked up or sent.

![Gratisaftehalen website](/images/portfolio/gratisaftehalen/main.png){.rounded-lg height="600" width="1000"}

## Key Features

- **Supper fast searching**: Find ads by filtering on categories, keywords and content
- **Saved searches**: Create searches using keywords and location, and notofy the user buy email
- **Messages**: Send and recieve messages 
- **Portal**: Overview of current ads and the reactions to these ads

## Impact

Gratisaftehalen.nl has helped keep the environment clean by reusing products, more than **200.000** users have either given away a product or recieved a free product.

![Gratisaftehalen website](/images/portfolio/gratisaftehalen/1.png){.rounded-lg height="600" width="1000"}

## Technologies Used

- **Frontend** : Razor pages
- **Frontend user Portal**: Durandal, Knockout
- **Backend**: Asp.Net Core 9.0, C#, ServiceStack, OrmLite
- **Cdn**: Our own highly optimized CDN for uploading and downloading images
- **Database**: Postgres, Redis
