# Markanda.nl - Free online marketplace

Markanda.nl is an innovative marketplace platform that connects businesses and indiviuals with customers, in order to buy, sell or give away produtcts.

![Markanda website](/images/portfolio/markanda/main.png){.rounded-lg height="600" width="1000"}

## Key Features

- **Product Catalog**: Beautiful product galleries with high-quality images and descriptions
- **Supper fast searching**: Find ads by filtering on categories, keywords and content
- **Bidding system**: And advanced bidiing system ensures teh right price
- **Saved searches**: Create searches using keywords and location, and notofy the user buy email
- **Messages**: Send and recieve messages 
- **Portal**: Overview of current ads and the reactions to these ads

![GratisaMarkanda berichten](/images/portfolio/markanda/2.png){.rounded-lg height="600" width="1000"}

## Impact

Markanda is a cooperation with [gratisaftehalen.nl]. More than 100.000 products have been sold using this platform.

![Markanda product page](/images/portfolio/markanda/1.png){.rounded-lg height="600" width="1000"}

## Technologies Used

- **Frontend** : Razor pages
- **Frontend user Portal**: Durandal, Knockout
- **Backend**: Asp.Net Core 9.0, C#, ServiceStack, OrmLite
- **Cdn**: Our own highly optimized CDN for uploading and downloading images
- **Database**: Postgres, Redis
